Has Financial-Aid Been Affected For New Yorker's Going to College Since COVID?

New York's Tuition Assistance Program (TAP) is NY’s largest student financial aid grant program which helps NY residents attending in state higher education institutions.Using state data to compare the pre-pandemic years (2018-2019) with the post- pandemic landscape (2022-2023), I analyzed which college sectors—public, private, or for- profit—saw the most significant changes in TAP funding and the number of student recipients.
